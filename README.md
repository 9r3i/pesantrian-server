# pesantrian-server
Server Library for Pesantrian

# requirements
this library requires and works upon the other libraries
- php server v8.1 or higher
- ai system -- [AI System](https://github.com/9r3i/ai) v4.7.1 or higher
- mserver v1.9.6 or higher
- eva framework for server v1.0.5 or higher
- ldb database driver v1.1.4 or higher

## more helpful library
- vcos (version control) v1.4.1 or higher
- gload v2.2.2 or higher
- eva admin client (for client maintenance)
- explorer (for client maintenance)

# files location
```
public_html/
- ai/
- db/
- libs/
  - pesantrian.php
  - pesantrianget.php
- root/
- server/
```
put the library files in the ```libs/``` directory, and will be loaded automatically as its called.
